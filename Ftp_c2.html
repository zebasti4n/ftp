¿Qué es Ftp?

FTP son las siglas de File Transfer Protocol (protocolo de transferencia de archivos). Es básicamente, un “protocolo” es un conjunto de procedimientos o reglas que permiten que los dispositivos electrónicos se comuniquen entre sí.FTP es el conjunto de reglas que los dispositivos de una red TCP/IP (Internet) utilizan para transferir archivos. Cuando usas Internet, en realidad utilizas una variedad de diferentes protocolos.

[*]El FTP es un protocolo cliente-servidor, en donde el cliente solicita los archivos y el servidor los proporciona. Como tal, el FTP necesita dos canales básicos para establecer una conexión: el canal de mando (inicia la instrucción, lleva la información básica, es decir, a qué archivos se debe acceder) y el canal de datos (transfiere los datos de los archivos entre los dos dispositivos). Para establecer una conexión, los usuarios deberán proporcionar credenciales al servidor FTP, que suele utilizar el puerto número 21 como modo de comunicación de forma predeterminada.

[*]En el modo activo, el servidor asume un rol activo aprobando una solicitud de datos.
[*]En el modo pasivo, el servidor no mantiene activamente la conexión, lo que significa que el usuario establece tanto el canal de datos como el canal de mando.
